						OVERVIEW
The Expense Tracker is a full-stack desktop application built using JavaFX for the frontend and Spring Boot for the backend.
It allows users to manage their daily expenses, track spending patterns, and securely authenticate into the system.
This project demonstrates full-stack development, REST API integration, layered architecture, and backend validation.

						Tech Stack

Java 17+
JavaFX
Spring Boot
Maven
MySQL(replace with what you used)
REST APIs
Layered Architecture (Controller → Service → Repository)

						Features

User registration & login validation
Secure authentication handling
Add new expenses
Edit existing expenses
Delete expenses
View all expenses
Categorize expenses
Real-time UI updates via API integration
Backend validation for login and input fields

						 Architecture

This project follows a clean layered architecture:

Controller → Service → Repository
Controller Layer – Handles incoming HTTP requests
Service Layer – Contains business logic
Repository Layer – Handles database interaction
This separation ensures maintainability, scalability, and clean code structure.

						Authentication

Login validation implemented
Backend validation for incorrect credentials
Structured request/response handling using JSON
